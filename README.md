# 💜 Portfolio de Jinnie, a Desenvolvedora da Criação 💻🎨

Bem-vinda(o) ao meu portfólio profissional! Este projeto foi desenvolvido para apresentar minhas habilidades como **Desenvolvedora** e **Ilustradora Digital**.

Aqui, você encontrará a união da minha paixão por interfaces intuitivas (código) e pela arte vibrante (design), resultando em projetos que são tanto funcionais quanto visualmente atraentes.

## ✨ Tecnologias Utilizadas

Este projeto foi construído utilizando as seguintes tecnologias principais no Front-end:

* **React:** Biblioteca JavaScript para construção de interfaces de usuário dinâmicas e base do projeto.
* **JavaScript (ES6+):** Linguagem de programação essencial para a interatividade e lógica.
* **HTML5 & CSS3:** Fundamentos para a estrutura e o estilo do projeto.
* **CSS Puro & Flexbox/Grid:** Utilizado para a estilização personalizada e responsividade.

## ⚙️ Como Executar o Projeto Localmente

Para rodar este portfólio no seu computador, siga estes passos simples:

1.  **Clone o Repositório:**
    ```bash
    git clone [https://github.com/AdrielleJcds/Portfolio.git](https://github.com/AdrielleJcds/Portfolio.git)
    ```

2.  **Navegue até o Diretório:**
    ```bash
    cd Portfolio
    ```

3.  **Instale as Dependências:**
    ```bash
    npm install
    ```

4.  **Inicie o Servidor de Desenvolvimento:**
    ```bash
    npm run dev
    ```
    O projeto será aberto no seu navegador, geralmente em `http://localhost:5173`.
